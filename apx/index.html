AlbionPacketExplorer

Avalonia desktop tool for decoding and exploring Albion Online network packets. After the first install it keeps itself up to date automatically.

§ 01

Download

Pick your platform. Apps update themselves after first install.

Installers are mirrored here from the project's release pipeline; if a download 404s, that release has not been mirrored yet (it retries automatically). The apps are not code-signed: a signing certificate is not worth the cost for a small free tool, so Windows SmartScreen or macOS Gatekeeper may warn on first launch. It is safe to allow it.

§ 02

Auto-update

Install once, then it keeps itself current.

After the first install, AlbionPacketExplorer checks for updates on its own and installs them in the background. No account, no sign-in, nothing to configure.
